Compass Pro Bono — NLP Survey Analysis Pipeline

Built an end-to-end NLP analytics pipeline to analyze open-ended survey responses from Compass Pro Bono clients and volunteers. The project cleans raw text data, detects sentiment, extracts recurring themes, and transforms unstructured feedback into actionable business insights using Python, AWS Comprehend, AWS S3, and Power BI.

Business Impact: Organizations often collect large amounts of qualitative survey feedback, but manually reading every response is slow and inconsistent. This pipeline automates analysis and helps decision-makers quickly identify sentiment trends, recurring concerns, and improvement opportunities.

FEATURED PROJECT

Fingertip Screening Device

Designed and built an end-to-end health screening device with a partner for a systems engineering project. The device detects hand proximity using an ultrasonic sensor, reads fingertip temperature through a thermistor and ADC conversion, displays the temperature on a 4-digit LED display, and provides LED-based positioning feedback. The complete system runs on a Raspberry Pi Pico using MicroPython.

Mission Context: Built against measurable mission requirements using a full Concept of Operations (CONOPS) and System Requirements Document (SRD). Every hardware and software decision traced back to system requirements.

Raspberry Pi Pico
